Commit Reminder

This App was created to help remind you to Commit Your Work

Too many of us forget to commit our work at correct intervals, so I built this app to remind you. It's going to be annoying, but it's for your own good:]

In Case You Forgot

  • ADD - "git add {specific path}" or "git add ." to add all changed file to staging
  • COMMIT - "git commit -m "{your messege here}", don't forget to write a description for your commit, it will help you and others find a way through all the different commits
  • PUSH - "git push" to push your work to the corresponding repository. if you don't have one yet, git will guide you on how to create one!
